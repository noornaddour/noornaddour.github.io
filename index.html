US Tariffs: IEEPA tariff refunds

What businesses need to know about current tariff refund mechanisms and investigations.

What you need to know

Recent tariff changes under IEEPA have created new refund considerations for importers.

  • 10% reciprocal tariffs in effect
  • Refund period up to 150 days
  • Investigations ongoing

Refund process

Importers must file adjustment requests with supporting documentation.
